# 개인정보 처리 안내 (founder-portfolio Skill)

**문서 버전: 2.0** · 발효일: 2026-05-06 · 이전 버전: 1.0 (fal.ai 수신자, 2026-05-06 폐지)

본 문서는 `founder-portfolio` 스킬이 사용자의 얼굴 사진을 외부 AI 이미지 생성 서비스(**OpenAI**)에 전송하여 처리할 때 적용되는 개인정보 처리 사항을 설명합니다. 이 흐름은 한국 「개인정보 보호법」(이하 "법") 제23조(민감정보 처리)와 제28조의8(개인정보 국외 이전)에 따른 별도 동의 항목입니다.

본 동의는 **`scripts/generate_portrait.py` 명령을 실행할 때만 적용**되며, 사진 없이 PDF만 렌더링하는 기본 사용(`scripts/render.py`)에는 적용되지 않습니다.

---

## 1. 처리하는 개인정보 항목

| 구분 | 항목 |
|------|------|
| 민감정보 | 사용자가 업로드한 얼굴이 포함된 사진 (생체인식정보) |
| 자동 수집 | 동의 일시(ISO 8601), 동의 약관 버전, 호스트 머신 식별자(`hostname`) |

본 스킬은 위 외의 식별정보(이름, 연락처, IP 주소 등)를 별도로 수집하지 않습니다. `profile.yaml`에 입력하는 이름·이메일·전화는 사용자가 직접 입력한 값으로, 본 동의의 범위 밖에서 사용자 머신에 저장됩니다.

## 2. 처리 목적

업로드된 얼굴 사진은 **AI 이미지 생성(스타일 변주 5종 — editorial / corporate / cinematic / monochrome / warm) 외 다른 목적으로 사용되지 않습니다.** OpenAI 공식 정책(2023-03-01 이후 default)에 따라 API 입력 데이터는 모델 학습에 사용되지 않습니다.

## 3. 보유 및 이용 기간

| 위치 | 보유 기간 |
|------|----------|
| 사용자 머신 (`assets/photos/`, `generated/`) | 사용자가 삭제할 때까지. `scripts/forget_me.py`로 즉시 일괄 삭제 |
| OpenAI 서버 | 최대 30일 (abuse monitoring 목적, OpenAI 공식 default policy). 학습에 사용되지 않음. Enterprise/ZDR 계정은 0일 |
| 동의 기록 (`.consent.json`) | 3년 (법 제29조 기록 보관 요건) — 보관 종료 시 자동 만료 |

## 4. 개인정보 국외 이전 (법 제28조의8 고지 사항)

`scripts/generate_portrait.py` 실행 시 사용자의 사진은 다음과 같이 국외로 이전됩니다.

| 항목 | 내용 |
|------|------|
| 이전받는 자 (수령인) | OpenAI, OpCo LLC |
| 연락처 | privacy@openai.com · https://privacy.openai.com |
| 이전 국가 | 미국 (United States) — Azure US datacenters |
| 이전 일시·방법 | 사용자가 명령을 실행하는 즉시, HTTPS API 호출 (POST `/v1/images/edits`) |
| 이전 항목 | 위 1항의 얼굴 사진 |
| 이전 목적 | AI 이미지 생성 (gpt-image-2 모델, image-to-image 변환) |
| 보유·이용 기간 | 위 3항 OpenAI 서버 행과 동일 |
| 거부 권리 | 동의를 거부할 수 있으며, 거부 시 본 기능을 사용할 수 없습니다. (PDF 렌더 등 다른 기능은 영향 없음.) |

## 5. 정보주체의 권리

사용자는 법 제35조부터 제37조까지에 따라 다음 권리를 가집니다.

- **열람**: `~/.claude/skills/founder-portfolio/.consent.json` 파일을 직접 확인할 수 있습니다.
- **정정·삭제**: `python scripts/forget_me.py` 명령으로 동의 기록과 본 머신에 저장된 사진(원본 + 생성물)을 일괄 삭제할 수 있습니다.
- **처리정지**: 위 명령을 실행하면 추가 처리가 즉시 중단됩니다.
- **OpenAI 서버에 남은 사본 삭제 요청**: https://privacy.openai.com 의 DSAR(Data Subject Access Request) 폼 또는 dsar@openai.com 으로 직접 요청하시기 바랍니다. 본 스킬은 OpenAI 서버의 데이터를 원격으로 삭제할 수 없습니다.

## 6. 동의의 효력

- 본 동의는 **약관 버전 2.0**에 대한 것입니다. 본 문서가 개정되면 다음 실행 시 자동으로 만료되며, 새 버전에 대한 재동의가 필요합니다.
- 약관 버전 1.0 (fal.ai 수령인, 2026-05-06 발효 후 동일자 폐지)은 본 버전 발효와 동시에 폐지됩니다.
- 동의 기록은 사용자의 머신에만 저장되며, 외부로 송신되지 않습니다.

## 7. 사용자 책임 영역 (gpt-image-2 회색지대 고지)

OpenAI 사용 정책은 "동의 없이 실제 인물의 photorealistic 이미지를 생성하는 행위"를 명시 금지합니다. 본 스킬은 **사용자 본인의 사진을 사용자 본인의 OpenAI API 키로 변환하는 self-use 케이스**에 한정하여 사용을 권장합니다.

- 타인의 얼굴 사진을 그 사람의 동의 없이 업로드하는 행위는 OpenAI 정책 위반입니다. 본 스킬은 이를 기술적으로 차단하지 않으므로, 위반 책임은 사용자가 부담합니다.
- OpenAI는 self-use 케이스를 정책 문언에 명시 분리하지 않은 회색지대로, 향후 정책 강화 시 계정 제재 가능성이 존재합니다. 사용자는 이 리스크를 인지하고 자기 책임 하에 사용합니다.
- 우려 시 회피 옵션: `--presets editorial,monochrome` 등 변환 횟수 최소화 + 결과 PNG만 보존, 입력 사진은 OpenAI 30일 보관 정책에 따라 자동 만료.

## 8. 책임 한계

본 스킬은 사용자가 직접 발급한 OpenAI API 키를 사용하여 사용자의 머신에서 실행됩니다. 본 스킬의 배포자는 사용자의 사진이 외부 서비스로 이전된 이후의 처리에 대한 직접적 통제권이 없으며, OpenAI의 약관·개인정보처리방침이 적용됩니다. 그러나 본 스킬을 설계·배포하는 행위 자체는 법 제2조(개인정보처리자) 정의 범위에 일부 해당하므로, 본 문서를 통해 정보주체에게 처리 사항을 명확히 고지합니다.
