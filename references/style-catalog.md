# Style Catalog — 35 Editorial References (v6 → v13)

> A founder portfolio's authority comes from restraint. Across 13 design cycles (v2–v13), 65 designs were produced. v6 onward consolidated into 35 distinct editorial references — each one a deliberate borrow from a publication whose visual language already carries credibility. This catalog is the "menu": pick the reference whose readership most closely matches the audience you are sending the PDF to.

The 5 Swiss Editorial variants in v13 are the **shipped MVP**. The other 30 entries are documented references — they were prototyped during v6–v12 and can be ported into `templates/` on demand.

## v13 — Shipped MVP (current base)

Five Swiss Editorial variants. **Same 12-column grid, same 8-section structure, same Inter / Source Serif Pro / Nanum Myeongjo typography.** Only the accent color and a few hairline elements vary. This is intentional: switching styles is a one-line change and the data file stays portable.

| ID | Status | Accent | Best Use |
|----|--------|--------|----------|
| `swiss-pure` | ✅ SHIPPED | Black/white only | Universal default. Maximum readability. Send when you don't know the recipient. |
| `swiss-red` | ✅ SHIPPED | Single red hairline #C8102E | Slight emphasis without breaking restraint. Press-kit, media outreach. |
| `swiss-cream` | ✅ SHIPPED | Black on cream paper #F5F0E4 + EB Garamond pairing | Magazine warmth. Editorial collaborations, publishing pitches. |
| `swiss-navy` | ✅ SHIPPED | Navy #0A2540 | B2B finance, formal proposal context, professional services. |
| `swiss-seal` | ✅ SHIPPED | Vermilion 朱印 #9F0824 + Korean seal overlay | Korean institutional contexts, legal-adjacent, government, university. |

## Porting a catalog entry into a shipped template

When asked to render a non-shipped style, the model should:

1. Locate the original sample at `/Users/sangrok/Library/Mobile Documents/com~apple~CloudDocs/Desktop/QJC/01_비즈니스/고객프로젝트/사용성연구소_이승필/samples_v{N}/sample_v{N}_{NN}_{name}.html`
2. Copy the closest shipped Swiss variant as a base: `cp templates/swiss-pure.html.j2 templates/{new-id}.html.j2`
3. Migrate only the accent color, font-family overrides, and any single decorative rule (border frame for TIME, gold rule for Wallpaper, etc.) — keep the 8-section grid intact.
4. Add `{new-id}` to the STYLES dict in `scripts/render.py` and test with `/founder-portfolio {new-id}`.

The 8-section structure is the constant. Style is the variable.
